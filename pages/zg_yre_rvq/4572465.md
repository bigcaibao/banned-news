### 特朗普指示推进对更多中国产品加征关税计划
------------------------

<div class="published">
 <span class="date" title="中国时间">
  <time datetime="2018-09-15T07:46:15+08:00">
   2018年9月15日 07:46
  </time>
 </span>
</div>
<br/>
<div class="wsw">
 <span class="dateline">
  华盛顿 —
 </span>
 <p>
  美国媒体报道说，虽然财政部长姆努钦试图重新启动与中国的贸易会谈，但总统特朗普已指示推进对额外2000亿美元中国产品加征关税的计划。
 </p>
 <p>
  彭博新闻社和路透社援引知情人士的话说，虽然财长姆努钦试图重启与北京的会谈，但特朗普总统希望推进加征关税的计划。
  <br/>
  <br/>
  有关报道使星期五的股市下跌，并导致人民币的币值下挫。
  <br/>
  <br/>
  白宫目前还没有对有关报道发表评论。
  <br/>
  <br/>
  彭博新闻社报道说，特朗普星期四与财政部长姆努钦、商务部长罗斯和美国贸易代表莱特希泽等最高贸易顾问讨论了关税问题。这次会议不在特朗普的公开日程安排上。
  <br/>
  <br/>
  在星期四的会议前，特朗普在推特上说，他没有感到要与北京达成协议的“压力”。他说：“他们有压力要跟我们达成协议。我们的市场在猛涨，他们的在崩溃。我们很快就要收到几十亿美元的关税并在国内制造产品。”
  <br/>
  <br/>
  对2000亿美元中国产品加征关税计划的公众评论期已在上星期结束。在这段期间，美国贸易代表办公室就关税计划收到了将近6千条评论。
  <br/>
  <br/>
  上星期，特朗普总统威胁要对另外2670亿美元的中国产品加征关税，这几乎是所有中国对美国的出口产品。
 </p>
 <p>
  他对记者说：“那样就改变了局势。”
  <br/>
  <br/>
  美国已经对价值500亿美元的中国货物加征了关税，中国也对美国货物进行了等量报复。
  <br/>
  <br/>
  特朗普政府说，对中国产品加征关税将迫使中国以更优惠的条件与美国进行贸易。
 </p>
 <p>
  特朗普政府还要求中国更好地保护美国知识产权，包括结束网络盗窃的做法。特朗普政府还呼吁中国扩大美国公司在中国的市场准入，并减少对美国的贸易顺差。
 </p>
 <p>
  中国已警告说，如果美国征收新的关税，中国也将予以报复。从美中两国贸易量的现状来看，中国已经没有多少等量报复美国制裁的余地。
 </p>
 <p>
 </p>
